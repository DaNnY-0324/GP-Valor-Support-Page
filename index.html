VL 100 Quick Start Guide

Welcome to EMV with GovtPortal! Let’s get your device(s) set up.

VL 100 Quick Start

Your device is automatically set up to plug and play for an ethernet connection, however, if you would prefer to use Wi-Fi, please follow these steps:

Once you're connected, your device will cycle to a screen labeled “Valor Connect” and state “Listening……..”

Your screen should always state “Valor Connect Listening…..” when not processing a transaction. If your machine is on the home screen:

  • Select the Favorites Button (★) on your Terminal
  • Select 10. Start VC to enable a Valor Connect connection on your VL100 or VL110

If you experience any issues with setup or maintaining a connection, please reach out to our support team via phone or email.

Transaction Process

Now that your machine is connected, when you select the Swipe/EMV button in your software to process a transaction, the machine will:

  • Light up and prompt the customer for confirmation of the transaction amount (green button).
  • Prompt the customer to swipe, dip, or tap their card.
  • Display the bank’s response as either Approved or Declined.
  • The software will reflect the bank’s response.

Please feel free to generate receipts via the software.

The device(s) will automatically batch all transactions processed that day.

Capture Preauth (Credit)

  1. Tap the Menu icon (☰).
  2. Tap Option 1 (Payment), tap on Ticket and enter the password.
  3. Select Tran Number or Card Number.
    • If Tran Number, enter the transaction number.
    • If Card Number, enter the last 4 digits of the card.
  4. Sign and Tip if prompted.
  5. Select receipt delivery method.

Pre-Sale Ticket

  1. Tap the Menu icon (☰).
  2. Tap the Page Down icon.
  3. Tap Option 2 (Host Utility).
  4. Enter the password and select Pre-Sale Ticket.
  5. Enter dollar amount, tap the OK button, and the pre-sale ticket will be printed.

Connect to WiFi

  1. From the Home Screen, tap (**★**) 7 Comm Config.
  2. Tap Comm Config, tap WiFi, tap the SSID, then tap Configure.
  3. Enter WiFi Password. (WiFi Password is case-sensitive).
  4. Tap OK to confirm, press the Cancel button, then tap Connect.

Favorites Menu

  • Reprint Receipt: Print receipt for the last transaction.
  • Settlement: Option to settle the current batch.
  • Tip Adjust: Adjust tip amount for transactions in the open batch.
  • Reports: View summary reports for open batch, last settled batch, and detailed reports for the last 5 batches settled.
  • Change Password: Set one general password or create custom passwords for settlements, voids, refunds, and removing custom fees.
  • Download Package: Applies parameter changes or updates the version on the device.
  • Comm Config: Configure your internet connection settings.
  • Reboot: Power cycles the device.
  • Remote Diagnostics: Allows ISO to remotely troubleshoot the device.

Transactions

Card Sale and Cash Sale (Dual Pricing)

  1. Enter Transaction Amount and press OK.
  2. Swipe/Insert/Tap card on the Present Card screen or start typing the card number for Manual Entry.
  3. Sign and Tip if prompted.
  4. Select receipt delivery method.

Credit Sale (Bin Based Surcharge)

  1. Enter Transaction Amount and press OK.
  2. Swipe/Insert/Tap card on the Present Card screen or start typing the card number for Manual Entry.
  3. Sign and Tip if prompted.
  4. Select receipt delivery method.
